I put this together in my free time while reading [Neural Networks and Deep Learning](http://neuralnetworksanddeeplearning.com/).

You can see the live preview [here](https://popboat1-nn-from-scratch.hf.space)

I wanted to actually look under the hood. Building the backprop loops, gradient descent, and loss functions entirely from scratch in plain Python and NumPy was a really fun way to figure out what's actually happening inside the black box.

You can see how to use the network through the notebook inside the src folder. Also, you can test the network by drawing through test_against_custom_drawing.py.
